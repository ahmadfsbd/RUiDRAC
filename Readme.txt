Pre-requisites!

1) Windows Platform
2) python 2.7 or later
3) Google Chrome

How to run?

1) Place the firmware file (.exe) in firmware folder

2) In the config.txt file, give full name of firmware file (eg example-123.exe) you want to install

3) Add/Delete server Credentials/Dictionaries in config.txt file (see example.config file)

Note: Enter url with ip and prefix (eg. http://192.168.10.10 https://192.168.10.10)

4) Run the update-firmware.py file by double clicking (Make sure python is added in environment variables)
